# 🤖 VK-бот «Моральный выбор»

## 📌 Описание проекта

**VK-бот «Моральный выбор»** — интерактивный бот, который предлагает пользователю различные жизненные ситуации, требующие нравственного выбора. Пользователь анализирует предложенные варианты действий, выбирает наиболее подходящий и получает объяснение, почему данный поступок считается нравственным или безнравственным.

---

## 🎯 Цель проекта

Разработка бота, который способствует:
- формированию моральных ценностей;
- развитию критического мышления;
- пониманию последствий своих поступков;
- воспитанию ответственности и уважения к окружающим.

---

## ⚙️ Основной функционал

### 📖 Прохождение моральных ситуаций

Бот предлагает пользователю различные жизненные ситуации и несколько вариантов ответа.

**Пример:**

> Вы нашли на улице кошелек с деньгами.

**Варианты действий:**
- Вернуть владельцу.
- Передать в полицию.
- Оставить себе.

После выбора бот объясняет последствия каждого варианта и оценивает его с точки зрения нравственных норм.

---

### 📝 Проверка знаний

Пользователь может пройти тесты на знание моральных принципов и получить итоговый результат.

---

### 📊 Статистика

После прохождения заданий отображаются:
- количество правильных ответов;
- процент нравственных решений;
- общий уровень прохождения.

---

## 🧠 Примеры ситуаций

| Ситуация | Нравственный поступок | Безнравственный поступок |
| :--- | :--- | :--- |
| Нашли чужую вещь | Вернуть владельцу | Забрать себе |
| Друг ошибся | Помочь исправить ошибку | Высмеять его |
| Стали свидетелем буллинга | Защитить пострадавшего | Пройти мимо |

---
